# CNN-based Cerebral Blood Flow Prediction

This project uses a **Convolutional Neural Network (CNN)** to predict cerebral blood flow patterns from simulated or experimental optical imaging data. The model is inspired by research in **biophotonics labs**, focusing on noninvasive monitoring of blood flow and oxygenation in tissues.

---

## Features

- End-to-end pipeline for loading data, training a CNN, and predicting cerebral blood flow maps.
- Visualization of results as heatmaps and curves.
- Compatible with Jupyter Notebook or Google Colab for easy experimentation.
- Ready for integration with rodent or human tissue imaging datasets.
